Third-Order Sudoku
==================

## Standard Form

0 表示需要求解的数字, 1-9 表示已有的数字

一行一行排列, 一共 81 个数字为一行, 用 LF 换行.

1. 要求解唯一
2. 旋转和镜像意义下同构


## Pipeline

获得一个数独, 用迭代器求出所有解, 如果有多个解标记为非法.

所有解求完, 计算解的旋转于镜像变换, 并作为签名.

接着合并多个数据库, 如果解同构, 取信息最少的那个.

导出数据, 计算问题难度.
